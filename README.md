# UMLP - BUT NFC 1 - TP web dynamique

## Bienvenue dans ce super module !

Ici, vous allez apprendre les rudiments de la programmation web. Vous découvrirez tout ce qu'il vous faut pour bien commencer votre parcours de développeur et appréhender l'environnement web. À la fin, vous serez capables de discuter avec des experts du domaine, de créer des produits simples et de comprendre les produits plus complexes.

### À quoi sert ce dépôt GIT ?

Pour les étudiants de 1ère année du BUT NFC RT de Montbéliard, il constitue l'unique moyen de diffusion des consignes de TP et des examens.

    - Consulter les sites de formation, forum, et autre ressources en ligne donnant du code tout fait.

### Utilisation de rt-projet

#### Uploader votre site sur le serveur

Pour uploader votre site, utiliser un client sftp, comme filezilla, en spécifiant :

- hôte : sftp://rt-projet-sftp.pu-pm.univ-fcomte.fr
- identifiant : votre login ENT.
- mot de passe : votre mot de passe ENT.
- port : 2222

#### Pour consulter votre site

Saisissez dans le navigateur https://rt-projet.pu-pm.univ-fcomte.fr/users/\[login\], où \[login\] est votre login ENT. Identifiez vous avec vos identifiants ENT.

#### Utiliser les bases de données.

1. Rendez-vous sur https://rt-projet.pu-pm.univ-fcomte.fr
2. Cliquez sur "Compte" en haut à droite.
3. Suivez la procédure pour activer votre **utilisateur de base de données**
    - Lors de vos connexions à la base de données via PHP, utiliser les informations fournies sur la page.
4. Cliquez sur "PHPMyAdmin" dans le menu principal.
