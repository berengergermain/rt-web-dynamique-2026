# UMLP - BUT NFC 1 - TP web dynamique

## Bienvenue dans ce super module !

Ici, vous allez apprendre les rudiments de la programmation web. Vous découvrirez tout ce qu'il vous faut pour bien commencer votre parcours de développeur et appréhender l'environnement web. À la fin, vous serez capables de discuter avec des experts du domaine, de créer des produits simples et de comprendre les produits plus complexes.

### À quoi sert ce dépôt GIT ?

Pour les étudiants de 1ère année du BUT NFC RT de Montbéliard, il constitue l'unique moyen de diffusion des consignes de TP et des examens.
